ChatList

Один промт — несколько нейросетей. Сравните ответы, сохраните лучшие, экспортируйте результат.

Скачать для Windows Исходный код

Возможности

  • Параллельная отправка промта в несколько моделей OpenRouter
  • Улучшение промта с помощью нейросети
  • Сохранение результатов в локальную SQLite-базу
  • Экспорт в Markdown и JSON
  • Тёмная и светлая тема
  • Работает из коробки — без настройки .env у пользователя

Установка

  1. Скачайте ChatList-*-Setup.exe (установщик) или chatlist-*.exe (portable).
  2. Запустите программу. Дополнительные файлы не нужны.
  3. При первом запуске рядом с exe создаётся папка data\ с базой данных.

Требования

Windows 10/11, 64-bit. Подключение к интернету для запросов к нейросетям.
